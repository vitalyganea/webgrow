## Usage

```bash
git clone https://github.com/vitalyganea/webgrow.gi

cd webgrow

composer install

cp .env.example .env

php artisan key:generate

npm install && npm run build

php artisan migrate

php artisan serve
```

> This project using Laravel + Laravel Breeze
